# 医疗护理系统 医生端（前端项目）


### front stack:

**js:**
 - es6 by babel
 - vue.js 2

**css:**
 - less
 - postcss 8(autoprefixer)
 - tailwindcss 2 with purgecss
 - element ui

**bundle tool:**
 - webpack 5


---


run command

```
npm install
```

to download dependencies.
then run

```
webpack serve
```

to launch the program
